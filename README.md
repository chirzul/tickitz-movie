# Tickitz Movie
Tickitz Movie is an backend API for movie-ticketing system built on Express.js

## Installation
1. Clone the repository
```bash
git clone https://github.com/chirzul/tickitz-movie.git
```
2. Install dependencies
```bash
npm install
```
3. Run the app
```bash
npm run start
```
:fire: You're all set!
